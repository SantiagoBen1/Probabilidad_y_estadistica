# Clasificación Sencilla y Vistosa con Scikit-Learn 🚀

Este repositorio/directorio contiene un Jupyter Notebook educativo diseñado para enseñar los fundamentos de la clasificación en Machine Learning usando `scikit-learn`. Aprenderás a cargar datos, crear *Pipelines*, buscar hiperparámetros, evaluar tu modelo y crear visualizaciones increíbles.
